# Meal Planner Blog

This repository contains my GitHub Pages blog for tracking the development of a Meal Planner app.

- Blog: [https://MouDas.github.io](https://MouDas.github.io)
- Homepage: `index.md`
- Blog posts stored in `_posts/`
- Daily progress updates in Markdown format

---

## How to Use This Blog

### 1. Clone the repository
```bash
git clone https://github.com/MouDas/MouDas.github.io.git
cd MouDas.github.io

2. Add a new post

Go to the _posts/ folder.

Create a new Markdown file using the format:
YYYY-MM-DD-title.md
Example: 2025-11-20-day-3-menu-planning.md

Include this front matter at the top of your post:

---
layout: post
title: "Day X: Your post title"
date: YYYY-MM-DD
---


Write your post in Markdown below the front matter.

3. Update the homepage (optional)

index.md already lists all posts automatically.

Only update index.md if you want to change the welcome text or layout.

4. Commit and push
git add .
git commit -m "Add Day X blog post"
git push
